# Git Comandos 
- git clone (clona um repositorio github para pc)
- git status(mostra o status do repositorio local)
- git add nome-arquivo (adicionar todos os arquivos do path no stage changes)
- git add . (Adicionar todos os arquivos do path no stage changes)
- git checkout -b (cria uma nova branch)
- git commit -m (salva os arquivos no repositorio local)
- git push (enviar os dados repositorio local para o repositorio remoto)
